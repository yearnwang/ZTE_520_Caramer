# ZTE_520_Caramer
进行中的项目......

从朋友手中得到了一个ZTE_C520的小兴看看，可惜的是，这个设备被其他手机绑定了。
他有一个后台系统，但是后台系统的密码需要在手机端才能查询的到。
也就是说必须有绑定的手机跟设备同时存在，才可以用此设备，否则光拿着设备是没用的。这点做的非常的好，提高了安全系数，值得我们借鉴。
